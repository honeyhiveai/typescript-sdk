## Overview

The HoneyHive SDK provides the following modules:

- **Session** - Start and manage sessions for tracking AI interactions
- **Events** - Log and retrieve events within sessions
- **Projects** - Manage HoneyHive projects
- **Configurations** - Handle prompt configurations and model settings
- **Datasets** - Create and manage evaluation datasets
- **Datapoints** - Work with individual data points in datasets
- **Experiments** - Run and analyze experiments
- **Metrics** - Define and track custom metrics
- **Tools** - Manage tool definitions for function calling
